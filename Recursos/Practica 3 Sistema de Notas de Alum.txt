Practica 3: Sistema de Notas de Alumnos

Pedir 20 notas finales de alumnos en una escala del 1 al 7, manejar decimales en las notas (doubles). Mostrar el promedio de las notas mayores a 5, ppromedio de notas inferiores a 4 y la cantidad de notas 1, ademas de mostrar el promedio final.

Ayuda: Usar un bucle for que itere 20(notas) y dentro del ciclo pedir las notas una a una para realizar calculos(contadores, sumas).

Opcional: si una de las notas ingresadas es 0 se debe salir del ciclo for y mostrar un mensaje de eeror finalizando el programa.
